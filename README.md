<img src="https://cdn.ebaumsworld.com/mediaFiles/picture/604025/85229439.jpeg">

3. Hannah Montana was originally called “Alexis Texas.”

<img src="https://cdn.ebaumsworld.com/mediaFiles/picture/604025/85229443.jpg">
